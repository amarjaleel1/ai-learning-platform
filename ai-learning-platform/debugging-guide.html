AI Learning Platform Debugging Guide

Important: This guide helps fix common issues with the platform, including navigation problems.

Common Issues and Solutions

1. Navigation Not Working

If you click on navigation items but nothing happens:

  • Make sure all script files are properly loaded (check browser console for errors)
  • Add the navigation-fix.js file to your index.html before the closing body tag
  • Ensure all data-nav and data-view attributes match correctly
Add this to index.html: 
<script src="js/navigation-fix.js"></script>

2. Application Gets Stuck on Loading

If the app loader/spinner doesn't disappear:

  • Check for JavaScript errors in the console
  • Make sure the loader has a timeout fallback
  • Add the following code to ensure the loader disappears:
// Add this to your main.js file
window.addEventListener('load', function() {
    setTimeout(function() {
        document.getElementById('app-loader').style.display = 'none';
    }, 2000); // Fallback timeout of 2 seconds
});

3. Content Not Displaying Properly

If content is invisible or improperly positioned:

  • Ensure CSS z-index values are set correctly (see browser-compatibility.css)
  • Check that the 'hidden' class is being toggled correctly
  • Make sure content containers have proper dimensions

4. Browser-Specific Issues

For issues that only appear in certain browsers:

  • Include the browser-compatibility.css file
  • Test in multiple browsers (Chrome, Firefox, Safari, Edge)
  • Use feature detection rather than browser detection
Add this to index.html: 
<link rel="stylesheet" href="css/browser-compatibility.css">

Specific Function Fixes

1. Code Editor Not Working

If the code editor doesn't respond or run code:

// Fix code editor focus issues
document.getElementById('code-editor').addEventListener('click', function() {
    this.focus();
});

2. Lesson Navigation Issues

If clicking on lesson items doesn't work:

// Manually trigger lesson click handling
document.querySelectorAll('.lesson-item').forEach(item => {
    item.addEventListener('click', function(e) {
        // Add click handler logic here
        const lessonId = this.getAttribute('data-lesson-id');
        console.log('Loading lesson: ' + lessonId);
        // Your lesson loading code
    });
});
